# Micromobility Data Analysis

## Overview
Analyzed ~1M scooter and bicycle trips to understand usage patterns and trends.

## Tools
Python, pandas, matplotlib, plotly

## What I Did
- Cleaned and processed raw data (dates, distances, durations)
- Filtered unrealistic values
- Created visualizations of monthly usage trends
- Explored how weather impacts trip duration

## Key Takeaways
- Usage increases in warmer months
- Data cleaning is critical for large datasets
- Visualization helps reveal patterns in dense data
